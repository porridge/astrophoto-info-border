# GIMP Astrophoto Info Border plugin

This is a from-scratch GIMP 3 compatible, Python reimplementation of the
[`border_information` plug-in](https://github.com/funxiun/gimp-plugin-astronomy/blob/master/scripts/border_information.scm)
from [the `gimp-plugin-astronomy` collection](https://github.com/funxiun/gimp-plugin-astronomy) for GIMP 2.

## Installation

1. `git -C $HOME/.config/GIMP/3.0/plug-ins clone https://github.com/porridge/astrophoto-info-border.git porridge-astrophoto-info-border`
2. Re-start GIMP


## Usage

![demo](pics/demo.gif)

1. Select the plugin from the new `Astrophotography` menu.
2. Set the text and other parameters as needed
3. Select `OK` and wait a bit for the plug in to generate an info border layer
